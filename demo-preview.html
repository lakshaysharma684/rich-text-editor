Style Allowlist

When pasting, only the following CSS properties are preserved to ensure your application's design system is respected:

  • Text elements: color, background-color, text-align, font-weight, font-style, text-decoration, vertical-align
  • Structural elements (img, table, div, figure): all of the above + border, width, height, margin, padding, float, overflow, object-fit, aspect-ratio
