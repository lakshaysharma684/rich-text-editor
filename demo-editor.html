v1.3.0
Fix Smart Paste Sanitizer Paste from Google Docs, Word, or any website — font sizes, font families, and layout-breaking styles are stripped automatically. Your design system is always preserved.
New Ctrl+Shift+V — Plain Text Paste Use Ctrl+Shift+V (or Cmd+Shift+V on Mac) to paste content as plain text, stripping all formatting.
New onImageUpload Callback Pass an onImageUpload async function to upload images to your server instead of embedding them as base64.
New AutoSave Restore Banner When an unsaved draft is found, a confirmation banner appears — choose "Restore Draft" or "Discard" instead of silent overwrite.
Fix setFontSize() Fixed Font size is now applied via span[style] elements correctly, replacing legacy <font> elements.

Welcome to the Rich Text Editor v1.3.0!

  • Press Ctrl+Shift+V to paste as plain text
  • Drag and drop an image, or paste a screenshot
  • Use the font-size dropdown and verify it applies correctly
  • Type something, wait a second, then refresh — you'll see the restore banner
